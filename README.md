# AI Resume Shortlisting System

An AI-powered web application that automates the resume screening process by analyzing candidate resumes and matching them with job requirements. The system helps recruiters save time by filtering and ranking resumes efficiently.

## Features

* Upload resumes in PDF format
* Extract text from resumes automatically
* AI-based resume analysis
* Candidate shortlisting
* User authentication (Login & Signup)
* Responsive web interface
* Contact, About, Credits and Terms pages

## Tech Stack

* Python
* Flask
* HTML5
* CSS3
* SQLite
* Jinja2 Templates

## Installation

1. Clone the repository

```bash
git clone https://github.com/rizwansheikh4865-art/AI-Resume-Shortlisting-System.git
```

2. Open the project folder

```bash
cd AI-Resume-Shortlisting-System
```

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Run the application

```bash
python app.py
```
