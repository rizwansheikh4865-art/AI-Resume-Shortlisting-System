About AIHIGH

Smart Resume Screening with AI and OCR

AIHIGH is a modern resume shortlisting platform designed to help recruiters, colleges, and organizations evaluate candidates faster and more accurately.

What AIHIGH Does

AIHIGH compares resumes with a job description using natural language processing techniques. It reads PDF resumes, extracts content, applies OCR for scanned documents, and ranks candidates based on similarity score.

The system also identifies important skills found in each resume, highlights missing skills, and provides suggestions to improve candidate-job alignment.

Key Technologies Used

  • Flask for backend web development
  • PyPDF2 for PDF text extraction
  • pytesseract for OCR support
  • scikit-learn for TF-IDF and cosine similarity
  • HTML, CSS, and Jinja templates for responsive UI

Why This Project Matters

Resume screening can take significant manual effort. AIHIGH reduces that effort by automatically ranking candidates and summarizing gaps, helping decision-makers focus on the strongest profiles quickly.

Who Can Use It

This project is useful for HR teams, placement cells, educational institutions, startups, and any organization that wants a simple AI-powered shortlist system.
